Documentation

You can find the documentation here. Feel free to ask questions in the forum thread if the guide is not clear.

Features

  • Grid based battle system.
  • Support all type of sprites.
  • Customizable windows.
  • Customizable scopes, are of effects and actions.
  • Sequences system.
  • Projectiles system.
  • Terrain, mark and aura effects.

Demo

You can grab the demo files there. I suggest you to run it. It showcases most of the features. Try all the skills and take a look at the configuration file. Create a new MV project then past the downloaded folders (img, plugins, data) to the root of the project.

AoE Creator

You can download my scope and area of effect creator there. Or use this online tool

Click on a black cell to select or deselect it. The center can be deselected too. The save button creates data in the text box. Use it in the configuration file to make custom scopes. The load button create a scope in the display area based on the data in the text box.
